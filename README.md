# Awesome-Matrix-Calculator
Calculator that perform matrix calculations on any NxN matrix<br/>

**Implemented Operations :**<br/>
- Addition
    - 2x2
    - 3x3
    - generic
- Subraction
    - 2x2
    - 3x3
    - generic
- Scalar Multiplication
    - 2x2
    - 3x3
    - generic
- Multiplication<br/>
- Transpose<br/>
- Determinant <br/>
